## Como adicionar uma transcrição

1. Compartilhar o conteúdo da reunião com o agente (texto ou áudio).
2. O agente aplica o **Workflow de Ingestão de Fonte** e a **Skill de Resumo de Reunião**.
3. Anonimização é verificada antes de salvar.
4. Usuário aprova o arquivo antes da criação.
5. Após criação, prosseguir para destilação de conhecimento (opcional imediatamente).
