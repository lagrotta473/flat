# References — Material Bruto (Somente Leitura)

> **ATENÇÃO: Arquivos nesta pasta são imutáveis após criação.**
> O agente nunca apaga, sobrescreve ou edita arquivos aqui.
> Novos arquivos podem ser adicionados (com aprovação do usuário).
> Erros factuais são tratados com notas de errata na `/knowledge_base/`.

## Estrutura

```
references/
├── reunioes/          # Transcrições de reuniões com o professor
├── normas/            # Textos normativos (ISO 9001, ABNT, etc.)
├── material_professor/ # Slides, apostilas, materiais didáticos
├── reflexoes/         # Reflexões pessoais do usuário
└── literatura/        # Artigos, livros, referências externas
```

## Nomenclatura

Todos os arquivos seguem o padrão: `AAAA-MM-DD_tipo_tema.md`

## Cabeçalho Padrão de Arquivo

```markdown
# [Título Descritivo]

**Tipo:** reuniao | reflexao | norma | material | artigo
**Data:** AAAA-MM-DD
**Fonte original:** [descrição — sem dados sigilosos]
**Anonimizado:** Sim | Não aplicável
**Processado para knowledge_base:** Não | Sim (data: AAAA-MM-DD)
```
